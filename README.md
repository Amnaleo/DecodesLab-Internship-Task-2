# 🌼 Pollen Classification

This project uses machine learning to classify pollen samples based on their features. It demonstrates the complete ML workflow, including data preprocessing, model training, prediction, and performance evaluation.

## ✨ Features
- Data preprocessing
- Classification model
- Model training and testing
- Performance evaluation
- Prediction on unseen data

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 🚀 Getting Started

### Install dependencies
```bash
pip install -r requirements.txt
```

### Run the project
```bash
python main.py
```

## 📊 Project Workflow
- Load dataset
- Preprocess data
- Train classification model
- Evaluate model
- Generate predictions
